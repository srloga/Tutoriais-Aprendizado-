Color:Nome auto sugestivo, através dela posso alterar a coloração de qualquer item de minha página, digitando o nome da cor ou os respectivos códigos HEX ou RGB da tonalidade desejada.
Ex1: color: red; Ex2: color: #FG0000.

Background:Propriedade usada para alterar o fundo de qualquer parte de minha página, seja ela um todo, ou apenas uma parte da mesma, para tal, posso utilizar de uma cor ou uma imagem.
Ex1: background-color:white; Ex2: background-image:url("link");

Border:Outra propriedade com nome auto sugestivo, trata-se do estilo, ajustes e delimitações das bordas, caso queiramos usá-las.
Ex1: border:2px solid black; Ex2: border-radius:10px

Margin:Mais uma vez uma propriedade auto sugestiva, com ela podemos definiver a margem da página, não só na estrutura mas também a tudo que seja referente aos textos, imagens e a qualquer elemento inserido na página.
Ex1: margin: auto; Ex2: margin: 70px 150px 150px 70px

Padding:Esta propriedade é muito similar a margin, sendo o diferencial dela o preenchimento do conteúdo interno, ao invés de afastar as extremidades da página, o padding ajusta o espaço dentro da própria área delimitada.
Ex1: padding:50px

Text:Mais uma propriedades importantes no CSS é a formatação, decoração e edição de textos, usadas com o código:Text.
Já aprendi várias formatações no tutorial deHTML, entretanto, no CSS isto se expande a capacidade de editar vários textos com apenas um código, ou referência.
Ex1: text-align: center; text-decoration: none

Fonts:Seguindo com as propriedades temos agora as fontes do texto, também é auto sugestivo, para quem está habituado a trabalhar com office, sabe da importância de saber escolher uma boa fonte, por padrão, o editor já indica vários tipos de fontes que podemos usar, entretanto, sempre posso ir ao google fonts buscar fontes diferenciadas para o meu projeto e importá-las usando url e depois definindo a "familía da fonte", estarei usando aqui nesse site como exemplo para referência posterior.

Display:Agora entrando para uma das propriedades mais importantes do css que são os controladores de layouts, para começar devo entender que os navegadores interpretam alguns códigos como displays do tipo bloco, por exemplo: as tags (div,h1 a h6,p,form,header,footer e section), o contrário disto temos as tags com o display padrão inline, como por exemplo: as tags (a,span,images)
Lembrando que isto é uma pré-definição, posso alterar cada uma destas propriedades de display no css, uma destas propriedades inclusive chama-se display:none; que a principio não parece tão útil,
mas que mais para a frente pode ser algo muito interessante a se usar, tomar nota de não confundir display:none com visibility:hidden.
Outra propriedade bem daora é a display:inline-block, que é um meio termo entre esses dois tipos de displays, me dando uma característica de display inline e a capacidade de manipulação do display block.

Especificidade CSS:É o que usamos para indicar ao navegador qual efeito, estilo ou forma que será a prioridade para aquele elemento, página, imagem e etc..por norma a escala hierarquica é:
Tag > CSS Interno > CSS Externo.
Vale ressaltar que o indicador é muito importante, no caso de classes ou ids, para que a ordem de autoridade seja seguida: ID/Class > Indicador Comum (h1, ul, li e etc..), posso ser ainda mais específico ao citar classe + indicador, por exemplo: h1.exemplo, neste caso só o/os indicador(es) h1 da classe "exemplo" seriam estilizados com meu css, isto é útil para estilizar elementos únicos numa determinada div/classe/id sem ter que a obrigatoriedade de editar todos os elementos da mesma.

Regra: !ImportantServe para determinar para o navegador qual o elemento de nossa página é o mais importante no css, ignorando a sequência de importância que foi explicado em Especificidade e nos fazendo ter total controle sobre, para isto, basta introduzir !important a seguir do código css que desejamos, por exemplo:
Em um css externo, ao selecionar o seletor h1 e informar: color:green !important, não importa qual id/classe que seria a especificidade, com este código, consigo definir a prioridade eu mesmo.

Overflow:Propriedade usada para ajustar comportamento de elementos dentro de uma div com tamanho delimitado, por exemplo:

overflow: scroll;Ao indentificar que o conteúdo da div é maior do que o tamanho da mesma, adapta-o para uma caixa de rolamento para melhor visualização.

overflow: auto;Também faz a mesma coisa que o scroll, a diferença é que no overflow auto, a propriedade entende que só precisar criar a caixa de rolamento se for necessário, se adaptando a quantidade de texto, ou seja, se por acaso os elementos deixarem de exister, o ajuste é feito automaticamente.

PS:Esta propriedade só funciona se eu especificar uma altura máxima para a minha div, caso contrário, por padrão todo os elementos serão ajustados na div automaticamente conforme a necessidade
