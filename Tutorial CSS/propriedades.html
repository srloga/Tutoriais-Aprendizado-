Positions:Mais uma propriedade importante no css, positions, com ela podemos ajustar blocos e posicionamentos de divs e containers, basicamente, o código positions serve para adequar os elementos na tela de forma estilizada e uniforme, por padrão, tudo que eu inserir na página vem com a configuração static, a primeira forma de trabalhar com positions é usando position:relative; a partir daí posso dar coordenadas para o meu elemento, o informando onde quero que ele esteja na tela, mesmo que seja sobreposto a outros elementos (diferente da margem que "move" todo o conteúdo em bloco de uma vez) a partir daí vem vários outras funções para adequar meu elemento.
Ex1: positions:relative; z-index:-1; (serve para utilizar os elementos como camadas)

Efeito Gradiente:Efeito de transição de cores em um mesmo elemento, conhecido também como degradê de cores, dentro do css, pode ser feito incluindo cores e direções, o mais simples é o gradiente linear (degradê em linha) referenciado com o código background-image: linear-gradient(), dentro do parentese especifico as cores e direções do gradiente, ex: (to right, blue, red), significa que o gradiente será feito para a direita usando as cores azul e vermelho.
Comandos de Direção: * Simples: to right/left/top/bottom. * Diagonais:to bottom left/right, to top right/left.
Também posso direcionar a direção do meu gradiente linear através de graus, desta forma: (180deg), substituindo o número pelo ângulo desejado. outro tipo de gradiente é o radial,"radial-gradiente" ao invés de ser em linha, é feito em formato de circulo, funciona da mesma forma, única diferença é que no dragradê radial só podemos informar as cores desejadas, e a porcetagem das mesmas.
