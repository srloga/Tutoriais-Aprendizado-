Entendendo Media Queries

Isso muda o jogo!

Propriedade que controla o que acontece quando temos diferentes resoluções de tela, Media Queries se trata de responsabilidade responsiva, ou ajuste automatico do site para as mais diversas resoluções que existem, sendo as principais 5 (smartphone, tablet, laptop, desktop, tv), sempre importante lembrar e ter em mente na hora de criar um site que é muito mais fácil ajustar apenas a estrutura da página, ao invés de elemento por elemento, abaixo deixarei indicado o código padrão para ajuste dos mesmos:

• Smartphones (600px para baixo)
@media only screen and (max-width: 600px) {...}

• Dispositivos um pouco maiores, mas ainda pequenos (Pequenos Tablets e Smartphones + largos, 600px para cima)
@media only screen and (min-width: 600px) {...}

• Dispostivos Médios (Tablets deitados, 768px para cima)
@media only screen and (min-width: 768px) {...}

• Dispostivos Largos (laptops/desktops, 992px para cima)
@media only screen and (min-width: 992px) {...}

• Super Largos (Telas maiores laptops, desktops e até TVs, 1200px para cima)
@media only screen and (min-width: 1200px) {...}
